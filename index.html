MOTOGEAR JP

制作目的

バイクパーツ輸入販売サイトのUI/UX改善を目的としたリニューアルデザイン。情報量は豊富だが視認性が低く、ユーザーが求める情報に辿り着きにくい課題を解決するために制作。

工夫したポイント
  • ダークテーマを採用し、バイク・モータースポーツのブランドイメージと統一
  • IntersectionObserver APIを使ったスクロールフェードインアニメーション
  • CSS Grid・Flexboxを組み合わせた柔軟なレイアウト設計
  • ホバーエフェクトとグラデーションオーバーレイで視覚的な奥行きを演出
  • カテゴリーフィルター機能でユーザーの目的別閲覧をサポート
使用技術
HTML5 CSS3 JavaScript GitHub Pages Google Fonts IntersectionObserver API
レスポンシブ対応
  • 768px・480pxの2段階ブレイクポイントで対応
  • スマホ時はハンバーガーメニューに切り替え
  • グリッドレイアウトをモバイルで1カラムに自動調整
  • clamp()関数で文字サイズをビューポートに合わせて流動的に変化

CORPORATE SITE

制作目的

中小企業向けコーポレートサイトのUIデザイン。信頼感・清潔感を重視しながら、ブランドの個性を打ち出せるデザインを目指した。Figmaでワイヤーフレーム→プロトタイプ→実装の一貫したプロセスで制作。

工夫したポイント
  • Figmaでコンポーネントを設計し、デザインの一貫性を担保
  • 余白・タイポグラフィを意識した読みやすいレイアウト
  • CSSカスタムプロパティでカラーテーマを一元管理
  • ファーストビューのCTAボタンを目立たせてコンバージョンを意識
使用技術
Figma HTML5 CSS3 CSS Variables GitHub Pages
レスポンシブ対応
  • モバイルファーストで設計し、PC向けに拡張
  • ナビゲーションをスマホ時にドロワーメニューへ切り替え
  • 画像をobject-fit: coverで比率を保ちながらトリミング

BRAND VISUAL

制作目的

新規ブランドのビジュアルアイデンティティ構築。ロゴ・バナー・SNS投稿素材を一貫したトーン&マナーで制作し、ブランドの世界観を統一することを目的とした。

工夫したポイント
  • ブランドカラーとフォントを定義したスタイルガイドを先に作成
  • Illustratorでロゴをベクターで制作し、あらゆるサイズに対応
  • Photoshopでバナーのビジュアル合成・写真レタッチを実施
  • SNS各プラットフォームの推奨サイズに合わせてテンプレート化
使用技術
Adobe Photoshop Adobe Illustrator Figma
